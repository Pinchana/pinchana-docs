## One project, six entry points

Pinchana is in **Preview**. Select the entry point that corresponds to the task. The browser interface is intended for users. The API, DLP, development, and deployment sections are intended for integrators and operators.

## Official services

- Web application: [pinchana.cc](https://pinchana.cc)
- Public API: [api.pinchana.cc](https://api.pinchana.cc)
- Documentation: [docs.pinchana.cc](https://docs.pinchana.cc)
- Pinchana BOT: [t.me/pinchanabot](https://t.me/pinchanabot)

Source is maintained in the [API](https://github.com/Pinchana/pinchana-api), [Web](https://github.com/Pinchana/pinchana-web), [DLP](https://github.com/Pinchana/pinchana-dlp), and [Docs](https://github.com/Pinchana/pinchana-docs) repositories.
